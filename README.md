# FoxF Frontend

FoxF 的前端项目，使用 Vue 3 + TypeScript + Vite + DaisyUI 构建。

- 🎨 优化UI交互
  - 通知面板居中显示
  - 添加磨砂玻璃效果
  - 优化动画效果

## 技术栈
- Vue 3
- TypeScript
- Vite
- DaisyUI
- TailwindCSS
- Vue Router
- Pinia

## 开发

# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build
